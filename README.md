# 1. Getting started
ZipIt is separated into 3 parts:
|Class|Functionality|
|:---|:---| 
|UZipper|Zip files into a zip archive|
|UUnzipper|Unzip a zip archive|
|UZipLibrary|Uses the Zipper/Unzipper internally for easy to use functions|
